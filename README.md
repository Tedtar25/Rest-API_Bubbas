# Bubba's API

RESTAPI para un sistema de pedidos tipo "pick up" de un local de la universidad.  
Desarrollada con **Node.js**, **Express** y **MySQL** como parte de un proyecto final.

##Características Principales

- CRUD de usuarios, productos y pedidos
- Estructura modular (routes, services, config)
- Conexión con base de datos MySQL
- Manejo de errores y middlewares
- Compatible con Postman o app móvil Android

##Tecnologías Usadas

- Node.js
- Express
- MySQL
- Nodemon
- JavaScript (ES Modules)
